# Security Policy

## Supported Versions

This repository contains detection rules, not application software.
The `main` branch is always the supported version.

## Reporting a Vulnerability

If you discover a security vulnerability in the detection logic, rule
content, or CI pipeline of this repository, please **do not open a
public GitHub issue**.

Report it privately via [GitHub's private vulnerability reporting](https://github.com/John-Axe/detection-as-code/security/advisories/new).

You can expect an acknowledgement within 48 hours and a resolution or
status update within 7 days.
